# Programs for ENSDF users
by V.Zerkin, 2025-2026

## Content

- **[ensdf-plus](ensdf-plus)** /_java_/ translating ENSDF file to ENSDF+ interpreted
    - genarates interactive Html-file with level-gamma plots
    - shows description of Type for Continuation Records
- **[set-dates](set-dates)** /_bash-shell_/ - setting dates of ENSDF files
    - sets modification time of ENSDF files from the latest update date of Datasets
    - genarates summary of ENSDF files and distribution of Datasets by years
    - to be used to set real dates of ENSDF files downloaded from ENSDF Archives
